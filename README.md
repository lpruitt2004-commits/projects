# Projects Repository

A collection of development projects showcasing web applications, AI tools, and learning materials.

## üìÅ Projects Overview

### [raspi5Kit](./raspi5Kit/)
A React Progressive Web App for Raspberry Pi 5 with Fallout 76-style touch interface

**Features:**
- Touch-only interface with on-screen keyboard
- GPIO pin control (22 pins, BCM numbering)
- System monitoring (CPU, temperature, storage)
- Network status tracking
- PWA with offline support
- Green monochrome CRT aesthetic

**Tech Stack:** React 18, Vite, ES6+, PWA

### [ai-scripts](./ai-scripts/)
CLxL2 AI script generation tools using CodeLlama and Llama2

**Features:**
- Generate bash and Python scripts using AI
- Code review and improvement suggestions
- Script explanation and documentation
- Interactive CLI tools

**Tech Stack:** Python 3.12, Ollama, CodeLlama-13B, Llama2-13B

### [learnToCodePortfolio](./learnToCodePortfolio/)
Learning materials and curriculum for full-stack development

**Curriculum:**
1. Prep & Foundations - Command line, version control, JavaScript/Python fundamentals
2. Frontend - HTML, CSS, DOM manipulation, responsive design, React
3. Backend - Node.js, Express, REST APIs, databases, authentication
4. Fullstack Projects - MERN stack apps, RESTful blog, capstone project
5. Data Science Python - Pandas, data visualization, machine learning
6. Career & Agile - Resume/LinkedIn, job search, interviews, Agile/Scrum

## üöÄ Quick Start

### Prerequisites
- **raspi5Kit:** Raspberry Pi 5, Node.js, npm
- **ai-scripts:** Python 3.12+, Ollama with CodeLlama-13B and Llama2-13B models
- **learnToCodePortfolio:** Basic understanding of HTML, CSS, JavaScript

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/lpruitt2004-commits/projects.git
   cd projects
   ```

2. **Navigate to a specific project**
   ```bash
   cd raspi5Kit        # For the Raspberry Pi PWA
   cd ai-scripts       # For AI script generation tools
   cd learnToCodePortfolio  # For learning materials
   ```

3. **Follow the README in each project directory** for specific setup instructions
